## Technical Approach

DSC measures field dynamics across three dimensions:
- **ε (Ethics):** Alignment with stated values
- **γ (Goals):** Purpose coherence and beneficial outcomes  
- **κ (Contradiction):** System tension and misalignment pressure

With adaptive time-stepping (Δt_eff) that adjusts pacing based on system capacity.
